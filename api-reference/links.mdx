## List links

Retrieve a paginated list of your short links.

```
GET /api/v2/links
```

### Query parameters

<ParamField query="limit" type="number" default="10">
  Maximum number of links to return.
</ParamField>

<ParamField query="skip" type="number" default="0">
  Number of links to skip before returning results. Use with `limit` for pagination.
</ParamField>

<ParamField query="all" type="boolean" default="false">
  Return links from all users. Admin accounts only.
</ParamField>

### Response

<ResponseField name="limit" type="number">
  The limit applied to the query.
</ResponseField>

<ResponseField name="skip" type="number">
  The number of links skipped.
</ResponseField>

<ResponseField name="total" type="number">
  Total number of links matching the query.
</ResponseField>

<ResponseField name="data" type="object[]">
  Array of link objects.
  <Expandable title="Link properties">
    <ResponseField name="id" type="string">
      Unique identifier (UUID).
    </ResponseField>
    <ResponseField name="address" type="string">
      The short link slug or custom URL path.
    </ResponseField>
    <ResponseField name="link" type="string">
      The full shortened URL.
    </ResponseField>
    <ResponseField name="target" type="string">
      The destination URL the short link redirects to.
    </ResponseField>
    <ResponseField name="description" type="string">
      Optional description for the link.
    </ResponseField>
    <ResponseField name="password" type="boolean">
      Whether the link is password-protected.
    </ResponseField>
    <ResponseField name="banned" type="boolean">
      Whether the link has been banned.
    </ResponseField>
    <ResponseField name="visit_count" type="number">
      Total number of visits.
    </ResponseField>
    <ResponseField name="created_at" type="string">
      ISO 8601 timestamp of when the link was created.
    </ResponseField>
    <ResponseField name="updated_at" type="string">
      ISO 8601 timestamp of the last update.
    </ResponseField>
  </Expandable>
</ResponseField>

### Example

```bash
curl https://kutt.it/api/v2/links?limit=10&skip=0 \
  -H "X-API-KEY: your-api-key"
```

```json
{
  "limit": 10,
  "skip": 0,
  "total": 1,
  "data": [
    {
      "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
      "address": "my-link",
      "link": "https://kutt.it/my-link",
      "target": "https://example.com",
      "description": "Example link",
      "password": false,
      "banned": false,
      "visit_count": 42,
      "created_at": "2024-01-01T00:00:00.000Z",
      "updated_at": "2024-01-02T00:00:00.000Z"
    }
  ]
}
```

## Create a link

Create a new short link.

```
POST /api/v2/links
```

### Request body

<ParamField body="target" type="string" required>
  The destination URL to redirect to.
</ParamField>

<ParamField body="description" type="string">
  An optional description for the link.
</ParamField>

<ParamField body="expire_in" type="string" placeholder="2 minutes/hours/days">
  How long until the link expires. Specify a number followed by a unit: `minutes`, `hours`, or `days`. For example, `"2 days"`.
</ParamField>

<ParamField body="password" type="string">
  A password to protect the link. Visitors must enter this password before being redirected.
</ParamField>

<ParamField body="customurl" type="string">
  A custom slug to use instead of a randomly generated one.
</ParamField>

<ParamField body="reuse" type="boolean" default="false">
  If `true` and a link with the same target already exists, return the existing link instead of creating a new one.
</ParamField>

<ParamField body="domain" type="string">
  A custom domain to use for the short link. The domain must be added to your account first.
</ParamField>

### Response

Returns the created [link object](#list-links).

### Example

```bash
curl -X POST https://kutt.it/api/v2/links \
  -H "X-API-KEY: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "target": "https://example.com",
    "customurl": "my-link",
    "description": "Example link",
    "expire_in": "7 days"
  }'
```

```json
{
  "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "address": "my-link",
  "link": "https://kutt.it/my-link",
  "target": "https://example.com",
  "description": "Example link",
  "password": false,
  "banned": false,
  "visit_count": 0,
  "created_at": "2024-01-01T00:00:00.000Z",
  "updated_at": "2024-01-01T00:00:00.000Z"
}
```

## Update a link

Update the target URL, address, description, or expiry of an existing link.

```
PATCH /api/v2/links/:id
```

### Path parameters

<ParamField path="id" type="string" required>
  The UUID of the link to update.
</ParamField>

### Request body

<ParamField body="target" type="string" required>
  The new destination URL.
</ParamField>

<ParamField body="address" type="string" required>
  The new short link slug.
</ParamField>

<ParamField body="description" type="string">
  An updated description for the link.
</ParamField>

<ParamField body="expire_in" type="string" placeholder="2 minutes/hours/days">
  A new expiry duration. Specify a number followed by a unit: `minutes`, `hours`, or `days`.
</ParamField>

### Response

Returns the updated [link object](#list-links).

### Example

```bash
curl -X PATCH https://kutt.it/api/v2/links/a1b2c3d4-e5f6-7890-abcd-ef1234567890 \
  -H "X-API-KEY: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "target": "https://example.com/new-page",
    "address": "my-updated-link"
  }'
```

## Delete a link

Permanently delete a short link.

```
DELETE /api/v2/links/:id
```

### Path parameters

<ParamField path="id" type="string" required>
  The UUID of the link to delete.
</ParamField>

### Response

<ResponseField name="message" type="string">
  A confirmation message indicating the link was deleted.
</ResponseField>

### Example

```bash
curl -X DELETE https://kutt.it/api/v2/links/a1b2c3d4-e5f6-7890-abcd-ef1234567890 \
  -H "X-API-KEY: your-api-key"
```

```json
{
  "message": "Link deleted successfully."
}
```

## Get link stats

Retrieve visit statistics for a short link, broken down by time period and dimension.

```
GET /api/v2/links/:id/stats
```

### Path parameters

<ParamField path="id" type="string" required>
  The UUID of the link.
</ParamField>

### Response

In addition to the [link object fields](#list-links), the response includes the following statistics fields:

<ResponseField name="lastDay" type="object">
  Statistics for the last 24 hours.
  <Expandable title="properties">
    <ResponseField name="views" type="number[]">
      Array of view counts over the period.
    </ResponseField>
    <ResponseField name="stats" type="object">
      Breakdown of visits by dimension.
      <Expandable title="properties">
        <ResponseField name="browser" type="object[]">
          Visit counts by browser. Each entry has a `name` (string) and `value` (number).
        </ResponseField>
        <ResponseField name="os" type="object[]">
          Visit counts by operating system. Each entry has a `name` (string) and `value` (number).
        </ResponseField>
        <ResponseField name="country" type="object[]">
          Visit counts by country. Each entry has a `name` (string) and `value` (number).
        </ResponseField>
        <ResponseField name="referrer" type="object[]">
          Visit counts by referrer. Each entry has a `name` (string) and `value` (number).
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="lastWeek" type="object">
  Statistics for the last 7 days. Same structure as `lastDay`.
</ResponseField>

<ResponseField name="lastMonth" type="object">
  Statistics for the last 30 days. Same structure as `lastDay`.
</ResponseField>

<ResponseField name="lastYear" type="object">
  Statistics for the last 365 days. Same structure as `lastDay`.
</ResponseField>

<ResponseField name="updatedAt" type="string">
  Timestamp of when the stats were last updated.
</ResponseField>

### Example

```bash
curl https://kutt.it/api/v2/links/a1b2c3d4-e5f6-7890-abcd-ef1234567890/stats \
  -H "X-API-KEY: your-api-key"
```

```json
  "lastDay": {
    "views": [0, 1, 3, 2],
    "stats": {
      "browser": [{ "name": "Chrome", "value": 5 }],
      "os": [{ "name": "Windows", "value": 4 }],
      "country": [{ "name": "US", "value": 6 }],
      "referrer": [{ "name": "google.com", "value": 3 }]
    }
  },
  "lastWeek": { "views": [], "stats": { "browser": [], "os": [], "country": [], "referrer": [] } },
  "lastMonth": { "views": [], "stats": { "browser": [], "os": [], "country": [], "referrer": [] } },
  "lastYear": { "views": [], "stats": { "browser": [], "os": [], "country": [], "referrer": [] } }
}
```
