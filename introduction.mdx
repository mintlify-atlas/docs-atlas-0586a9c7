## What is Kutt?

Kutt is a modern URL shortener you can host yourself. Create short links, attach them to your own custom domains, track click statistics, and manage everything through a clean web UI or the REST API.

## Supported databases

Kutt works with three database backends — no separate setup needed for the default SQLite option:

| Database | Driver | Notes |
|---|---|---|
| SQLite | `better-sqlite3` | Default — no extra service needed |
| PostgreSQL | `pg` | Recommended for production |
| MySQL / MariaDB | `mysql2` | Full feature parity |
